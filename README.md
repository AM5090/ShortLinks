1. Нужно ввести ссылку для сокращения либо ввести букву L 
   для просмотра доступных ссылок.
2. Если ввели ссылку нужно указать количество кликов для ссылки.
3. Если ссылка уже есть у данного пользователя, то будет выведена 
   информация по данной ссылке.
4. Если ссылка есть у других пользователей, то будет короткая ссылка 
   будет сформирована с большим количеством символов для избежания повторений.
5. После вывода всего списка доступных ссылок будет ожидаться ввод id 
   одного из ссылок. Список выводится доступный только текущего пользователя.
6. Если ссылка доступна, то она будет открыта в браузере.
7. Если ссылка не доступна, то пользователь будет проинформирован.
